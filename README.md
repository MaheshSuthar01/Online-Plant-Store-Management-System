Online Plant Store Management System: 
•	Online Plant Store Management System is a full-featured eCommerce web application developed to streamline the online sale and management of indoor and outdoor plants. 
•	The system is built using ASP.NET with C# for backend development, SQL for database management, and HTML, CSS, and JavaScript for the frontend. It provides a user-friendly interface for browsing products, managing carts, placing orders, and includes an admin panel for add new products and delete, edit products.
